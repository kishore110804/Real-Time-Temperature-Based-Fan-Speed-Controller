SIMULATION GUIDES (NEW - USE THESE)
===================================

1. PROTEUS_COMPLETE_GUIDE.txt
   → Complete step-by-step guide for Proteus circuit simulation
   → Includes component placement, wiring, and HEX file upload
   → Use this for hardware circuit simulation

2. RENODE_SIMULATION_GUIDE.txt  
   → Virtual hardware simulation (no physical devices needed)
   → Faster and easier than Proteus
   → FREE alternative to Proteus
   → Use this for quick software testing


SIMULATION SCRIPTS
==================

3. stm32_sim.resc
   → Main Renode simulation script
   → Loads STM32F103C8 and your firmware
   → Opens UART window automatically
   → Run with: renode stm32_sim.resc

4. temp_sweep.resc
   → Automatic temperature sweep (25°C to 100°C)
   → Simulates LM35 sensor input changes
   → Run inside Renode: include @temp_sweep.resc

5. simulate_complete.py
   → Python script for ngspice/KiCad simulation
   → Generates graphs and CSV data
   → Run with: python simulate_complete.py


SOURCE CODE FILES
=================

6. src/main.c
   → Main STM32 firmware code
   → Contains ADC, PWM, UART logic
   → Temperature-to-fan-speed algorithm

7. src/stm32_hal_init.c
   → HAL library initialization functions
   → System clock, GPIO, ADC, TIM3, UART1 config

8. include/main.h
   → Pin definitions and constants
   → Function prototypes

9. include/stm32_hal_init.h
   → HAL initialization function declarations

================================================================================
QUICK START - CHOOSE YOUR PATH
================================================================================

PATH 1: RENODE (Easiest - Recommended)
---------------------------------------
1. Install Renode from https://renode.io/
2. Build firmware: pio run -e bluepill_f103c8
3. Run simulation: renode stm32_sim.resc
4. Test temperature: include @temp_sweep.resc
5. Take screenshot of UART output


PATH 2: PROTEUS (Traditional Circuit Simulation)
-------------------------------------------------
1. Open PROTEUS_COMPLETE_GUIDE.txt
2. Follow PART 1: Build circuit (12 components)
3. Follow PART 2: Load firmware.hex
4. Follow PART 3: Run simulation
5. Take screenshot of working circuit


PATH 3: KICAD + NGSPICE (Circuit Analysis)
-------------------------------------------
1. Open RTS_FanControl.kicad_pro in KiCad
2. Run: python simulate_complete.py
3. View generated graphs and data

================================================================================
NEED HELP?
================================================================================

Renode not working?
  → Check RENODE_SIMULATION_GUIDE.txt troubleshooting section

Proteus not working?
  → Check PROTEUS_COMPLETE_GUIDE.txt troubleshooting section

Build errors?
  → Run: pio run -v (verbose mode)
  → Check platformio.ini configuration

Simulation not showing output?
  → Verify UART is configured in code (115200 baud)
  → Check HAL initialization in stm32_hal_init.c
